Project on Console

To set visual studio, you have to create a folder and open it using Visual Studio Code. Then, you have to click on the file icon to start writing code. You can also install shortcuts by clicking on the fourth icon on the right side.

To see the console screen you have to click on 'view', and then select terminal.The console screen is used so that you can see the output.You can print something on the console screen using console.log() .
